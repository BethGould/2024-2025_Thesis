This contains functions used by reference by the ODE solver. They are included outside the class structure and in a unified form so that they can be easily switched out.

Здесь содержатся функции, используемые решателем ODE по ссылке. Они включены вне структуры класса и в унифицированном виде, так что их можно легко отключить.

--DERIVATIVE FUNCTIONS FOR ODE SOLVERS--
x
y  = [psi, dpsi/dx]
k  -- electron wavenumber
B  -- magnetic field strength
R  -- radius -- note that here only B * R is relevant, not B and R independently
mu -- ee coupling strength
mA -- maximum amplitude -- used by events, not derivative solvers, but must be included

For the input to the loop, without magnetic field or coupling
Contains both versions together, for compactness.

xy = [psi, dpsi/dx]
